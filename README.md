## 프로젝트 소개
한 곳에 가입해 여러 전시를 한 눈에 보고 간단하게 예매할 수 있는 웹사이트

일반 회원도 전시를 등록할 수 있도록 해 등록&예매가 가능한 전시 플랫폼을 구현

</br>

## 기술 스택
html5, css, java, jsp, mysql

</br>

## 구현 기능
### 1. 로그인
DB 내 회원정보와 대조하여 인증
> id: taesub | password: taesub0731로 로그인 가능
### 2. 검색
전시회명, 장소에서 검색어를 포함하는 결과 조회
### 3. 예매
### 4. 전시 등록
### 5. 마이페이지
전시 예매/등록 조회 및 삭제, 비밀번호 변경
